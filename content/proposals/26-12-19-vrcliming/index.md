## Inmersive game development for students 

We would like to create the following environment. **Students create the games that students play.**
This time we will create a VR Climbing demo.

This forms a life cycle of learning and play. It is the starting point for game generation, where students learn to build immersive environments and design game competitions for their peers. In this way, we foster motivation and active engagement in the classroom.
